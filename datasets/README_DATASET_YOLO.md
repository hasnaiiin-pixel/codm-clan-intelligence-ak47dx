# AK47DX YOLO / OCR dataset

Questa cartella serve per preparare training futuro.

- `yolo/images`: screenshot originali.
- `yolo/labels`: label YOLO in formato `.txt`.
- `ocr/crops`: ritagli OCR usati per Kill / Death / Assist, mappa, data, profilo.
- `ocr/ground_truth.csv`: valore OCR e valore corretto manualmente.
- `exports`: pacchetti esportati per training.

Regola: ogni correzione manuale affidabile deve diventare un campione di training.
