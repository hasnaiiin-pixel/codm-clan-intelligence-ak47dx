# MVP 0.2 - Screenshot Stats Core

Versione sviluppata sugli screenshot CODM reali.

## Aggiunto
- Import profilo/statistiche: Multigiocatore, Battle Royale, Zombi, DMZ.
- Import partita migliorato: CED, Postazione/Hardpoint, Dominio.
- Campi partita: score, U/M/A, impatto, tempo, catturate, rapporto U/M, precisione, headshot.
- Matching nickname con roster per ridurre doppioni.
- Dashboard con K/D clan, win rate, ranking player.
- Loadout Center con screenshot, arma primaria/secondaria, specialità e gunsmith.
- Migrazione Supabase `supabase/migration_0_2.sql`.

## Da fare su Supabase
Se hai già la 0.1 funzionante, esegui una sola volta `supabase/migration_0_2.sql` in SQL Editor.
