# Release 2.0 Definitive AK47DX

## Decisione roadmap

La roadmap viene compressa in una sola major release: 2.0.

## Moduli inclusi

- Dashboard finale gaming
- Action Panel con screenshot inline
- Inviti e registrazione player tramite link/QR
- Clan HQ con storia, capi, vice, social e avvisi
- Archivio partite filtrabile
- Player manuali e registrati
- Associazione clan
- Ranking Gold/Silver/Bronze
- Kill / Death / Assist only
- Backend OCR Hybrid allineato
- Supabase migration 2.0
- Dataset YOLO-ready futuro

## Regola screenshot

Lo screenshot partita deve essere sempre consultabile dentro Action Panel. Non deve obbligare ad aprire una nuova pagina o tab.
