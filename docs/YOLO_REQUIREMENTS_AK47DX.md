# AK47DX - Cosa serve per YOLO reale

YOLO non si può rendere affidabile solo con codice. Serve dataset reale.

## Screenshot necessari

Minimo per iniziare:
- 100 screenshot CED.
- 100 screenshot Postazione.
- 50 screenshot profilo player.

Consigliato per un risultato serio:
- 300-500 screenshot per schermata/modalità.
- Screenshot da telefoni diversi.
- Screenshot WhatsApp e screenshot diretto CODM.
- Vittorie e sconfitte.
- Mappe diverse.
- Qualità diverse.

## Cosa bisogna etichettare

Per match:
- result/header.
- score blu/rosso.
- data/ora.
- modalità/mappa.
- tabella blu completa.
- tabella rossa completa.
- Kill/Death/Assist di ogni riga.
- nickname se serve.

Per profilo:
- nickname.
- UID.
- livello.
- leggendario MG.
- leggendario BR.
- leggendario DMZ.
- leggendario Zombie.

## Come l'app aiuta

La 2.0.1 salva:
- screenshot originale.
- box calibrazione.
- crop OCR.
- valore OCR.
- valore corretto manualmente.

Questi dati diventano dataset per training.
